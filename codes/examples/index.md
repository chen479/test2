### MATLAB FFT 示例

**示例目标**：  
演示如何使用 MATLAB 内置的快速傅里叶变换（FFT）函数对信号进行频谱分析。

**技术要点**：
- 生成包含两个频率分量的合成信号（例如 50 Hz 和 120 Hz）
- 计算信号的 FFT 并得到幅度谱
- 正确绘制频率轴（基于采样频率和信号长度）
- 理解频谱的对称性和单边谱的表示方法

**代码文件**：
- [fft_example.m](fft_example.m)（点击下载/查看）

### Python 正弦波生成示例

**示例目标**：  
使用 Python 的 NumPy 和 Matplotlib 库生成并可视化正弦波信号，演示基本信号生成和绘图方法。

**技术要点**：
- 设置采样频率、持续时间、信号频率等参数
- 使用 NumPy 生成时间轴和正弦波序列
- 使用 Matplotlib 绘制时域波形
- 理解采样定理的基本概念（采样率必须大于信号最高频率的两倍）

**代码文件**：
- [sine_wave.py](sine_wave.py)（点击下载/查看）
