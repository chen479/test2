# 语音降噪系统项目

## 项目目标

设计并实现一个基于数字信号处理的语音降噪系统。通过本项目，你将掌握噪声分析、降噪算法设计和实现技术。

## 技术要求

- 分析噪声特性
- 设计合适的降噪算法（如谱减法、维纳滤波器等）
- 实现降噪系统
- 评估降噪效果

## 设计步骤

1. **噪声分析**：采集或生成噪声信号，分析其频谱特性
2. **选择降噪算法**：根据噪声特性选择合适的降噪算法
3. **实现降噪系统**：编写代码实现降噪算法
4. **评估降噪效果**：使用信噪比等指标评估降噪效果
5. **优化算法**：根据评估结果优化算法参数

## 项目文件

- [`speech_denoise.m`](speech_denoise.m)：MATLAB实现
- [`speech_denoise.py`](speech_denoise.py)：Python实现

## 运行说明

### MATLAB版本
1. 打开MATLAB
2. 导航到项目目录
3. 运行 `speech_denoise.m` 文件
4. 观察输出结果和图形

### Python版本
1. 确保安装了必要的库：`numpy`、`scipy`、`matplotlib`、`soundfile`
2. 导航到项目目录
3. 运行 `python speech_denoise.py`
4. 观察输出结果和图形
