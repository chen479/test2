# 数字滤波器设计与实现项目

## 项目目标

设计并实现一个数字滤波器，满足给定的技术指标。通过本项目，你将掌握数字滤波器的设计流程、实现方法和性能分析技术。

## 技术指标

设计一个低通滤波器，满足以下指标：
- 通带截止频率：1kHz
- 阻带截止频率：2kHz
- 通带最大衰减：1dB
- 阻带最小衰减：40dB
- 采样频率：8kHz

## 设计步骤

1. **选择滤波器类型**：根据需求选择IIR或FIR滤波器
2. **确定滤波器阶数**：根据技术指标计算所需的滤波器阶数
3. **设计滤波器系数**：使用合适的设计方法（如巴特沃斯、切比雪夫等）
4. **实现滤波器**：编写代码实现滤波器
5. **分析滤波器性能**：绘制频率响应、冲激响应等
6. **验证滤波器效果**：用实际信号测试滤波器性能

## 项目文件

- [`filter_design.m`](filter_design.m)：MATLAB实现
- [`filter_design.py`](filter_design.py)：Python实现

## 运行说明

### MATLAB版本
1. 打开MATLAB
2. 导航到项目目录
3. 运行 `filter_design.m` 文件
4. 观察输出结果和图形

### Python版本
1. 确保安装了必要的库：`numpy`、`scipy`、`matplotlib`
2. 导航到项目目录
3. 运行 `python filter_design.py`
4. 观察输出结果和图形

## 扩展任务

2. 尝试不同的设计方法（如椭圆滤波器）
3. 实现实时滤波器
4. 比较IIR和FIR滤波器的性能差异
