# 第12章 典型应用

## 12.1 引言

数字信号处理技术已广泛应用于各个工程领域。本章选取几个典型应用领域进行介绍，包括语音信号处理、音频均衡器设计、生物医学信号处理、雷达与声呐信号处理，以及数字信号处理与机器学习的结合。通过这些实例，读者可以理解如何将前几章的理论知识应用于实际问题。

---

## 12.2 语音信号处理

### 12.2.1 语音信号的特点
语音信号是非平稳的随机信号，但在短时（10-30 ms）内可视为平稳。主要参数包括：
- **基音周期**：声带振动的周期，决定音高。
- **共振峰**：声道谐振频率，决定音色。
- **短时能量**、**短时过零率**等。

### 12.2.2 语音编码
语音编码的目的是在保证一定语音质量的前提下，用尽可能低的比特率表示语音。常见编码方式：
- **波形编码**：如 PCM、ADPCM，直接对波形进行量化和预测。
- **参数编码**：如 LPC（线性预测编码），提取语音参数（基音、共振峰、清/浊音标志）进行编码。
- **混合编码**：如 CELP，结合波形和参数编码，用于移动通信。

**LPC原理**：语音信号 $s[n]$ 可用全极点模型表示：
$$
s[n] = \sum_{k=1}^{p} a_k s[n-k] + G u[n]
$$
其中 $u[n]$ 为激励（清音为白噪声，浊音为周期脉冲），$a_k$ 为线性预测系数。通过求解自相关方程可得系数，进而实现低速率编码。

### 12.2.3 语音增强
语音增强的目标是从带噪语音中提取纯净语音。常用方法：
- **谱减法**：假设噪声平稳，从带噪语音幅度谱中减去噪声幅度谱估计。
- **维纳滤波**：在频域设计滤波器最小化均方误差。
- **子空间法**：将语音信号投影到信号子空间和噪声子空间。
- **深度学习方法**：使用神经网络直接映射带噪语音到纯净语音。

### 12.2.4 语音识别
语音识别将语音信号转换为文本。传统系统包括特征提取（MFCC）、声学模型（HMM）、语言模型等。现代系统多采用端到端深度学习模型（如CTC、Attention、Transformer）。

## 12.3 音频均衡器设计

### 12.3.1 均衡器的作用
音频均衡器用于调整不同频率成分的增益，以补偿音响系统或听音环境的缺陷，或实现特定音效。常见类型：
- **图示均衡器**：将频带划分为若干固定频段，每个频段用独立的增益控制。
- **参量均衡器**：每个频段可调中心频率、增益和品质因数 Q。

### 12.3.2 滤波器设计
均衡器的核心是带通/带阻滤波器组。对于图示均衡器，常用二阶 IIR 滤波器（如峰值滤波器）实现：
$$
H(z) = \frac{b_0 + b_1 z^{-1} + b_2 z^{-2}}{1 + a_1 z^{-1} + a_2 z^{-2}}
$$
通过调节系数可实现不同中心频率和 Q 值的增益调整。滤波器组需满足整体平坦响应（当所有增益为 0 dB 时，总响应为全通）。

### 12.3.3 实现示例
一个 10 段图示均衡器，中心频率按倍频程分布（31.25 Hz、62.5 Hz、125 Hz、…、16 kHz），每个频段用二阶峰值滤波器，设计系数可由公式计算或使用 MATLAB 的 `fdesign.parameq`。

**MATLAB 代码片段**：
```matlab
fs = 44100; % 采样率
fc = 1000;  % 中心频率
BW = 1;     % 带宽（倍频程）
G = 6;      % 增益 dB
[b, a] = designParamEQ('Bandwidth', BW, 'CenterFrequency', fc, ...
    'Gain', G, 'SampleRate', fs);
freqz(b, a, 2048, fs);
```

## 12.4 生物医学信号处理

### 12.4.1 心电图（ECG）信号处理
ECG 记录心脏电活动，典型波形包括 P 波、QRS 复合波、T 波。处理目标：
- **去噪**：消除工频干扰（50/60 Hz）、肌电干扰、基线漂移。
- **特征提取**：检测 QRS 波位置，计算心率，分析 ST 段变化。

**去噪方法**：
- 陷波滤波器滤除工频。
- 高通滤波器（截止频率 0.5 Hz）去除基线漂移。
- 低通滤波器（截止频率 40 Hz）去除高频噪声。
- 小波变换可有效保留 QRS 波特征。

**QRS 检测**：常用 Pan-Tompkins 算法，包括带通滤波、微分、平方、移动窗口积分和自适应阈值。

### 12.4.2 脑电图（EEG）信号处理
EEG 记录大脑电活动，频率分为 δ（0.5-4 Hz）、θ（4-8 Hz）、α（8-13 Hz）、β（13-30 Hz）、γ（>30 Hz）等节律。应用包括：
- **伪迹去除**：眼电、肌电干扰的去除（ICA、小波）。
- **事件相关电位（ERP）**：叠加平均提取刺激诱发的电位。
- **脑机接口（BCI）**：提取运动想象、SSVEP 等特征，控制外部设备。

### 12.4.3 医学成像
- **超声成像**：通过脉冲回波信号处理形成图像，包括波束形成、包络检测、对数压缩。
- **磁共振成像（MRI）**：原始数据为 k 空间数据，通过二维傅里叶逆变换得到图像。
- **CT 成像**：滤波反投影算法重建图像。

## 12.5 雷达与声呐信号处理

### 12.5.1 雷达原理
雷达发射电磁脉冲，接收目标回波，通过测量时延和频率变化确定目标距离和速度。基本信号处理包括：
- **脉冲压缩**：通过匹配滤波器提高距离分辨率。
- **动目标检测（MTI）**：利用多普勒效应滤除静止杂波。
- **恒虚警率（CFAR）检测**：自适应阈值检测目标。

### 12.5.2 脉冲压缩
发射线性调频信号（chirp）：
$$
s(t) = \text{rect}\left(\frac{t}{T}\right) e^{j2\pi(f_c t + \frac{1}{2}\mu t^2)}
$$
接收回波与匹配滤波器卷积实现脉冲压缩，压缩后脉冲宽度为 $1/B$（$B$ 为带宽），距离分辨率 $\Delta R = c/(2B)$。

### 12.5.3 合成孔径雷达（SAR）
SAR 通过平台运动合成大孔径，获得高方位分辨率。成像算法包括距离多普勒算法（RDA）、Chirp Scaling 算法（CSA）等，核心是二维匹配滤波。

### 12.5.4 声呐信号处理
声呐使用声波探测水下目标，信号处理类似雷达，但需考虑水声信道多途效应、环境噪声等。常用技术包括波束形成、匹配场处理等。

## 12.6 数字信号处理与机器学习

### 12.6.1 特征提取
许多机器学习任务以信号处理提取的特征作为输入：
- **时域特征**：均值、方差、过零率、峰值等。
- **频域特征**：频谱质心、谱熵、子带能量等。
- **倒谱特征**：MFCC（梅尔频率倒谱系数）广泛用于语音和音频分类。
- **小波特征**：小波包能量用于振动信号故障诊断。

### 12.6.2 深度学习与端到端处理
深度学习可直接从原始波形学习特征，替代传统特征提取+分类器的两阶段方法。典型模型：
- **卷积神经网络（CNN）**：用于音频场景分类、语音识别、ECG 分类。
- **循环神经网络（RNN/LSTM）**：处理时序信号，如语音识别、EEG 预测。
- **Transformer**：近年用于语音识别、音乐生成。

### 12.6.3 应用实例：基于深度学习的语音分离
语音分离（鸡尾酒会问题）将混合语音中的目标语音分离出来。典型模型：Conv-TasNet，使用时域卷积网络直接处理波形，在信噪比改善上超越传统方法。

数字信号处理的应用领域极其广泛，本章仅选取了语音、音频、生物医学、雷达和机器学习几个方向进行了简要介绍。通过本章的学习，读者应能理解基本理论与实际应用之间的联系，并具备将所学知识应用于简单问题的能力。数字信号处理仍在不断发展，与人工智能、物联网等技术的融合将催生更多创新应用。
