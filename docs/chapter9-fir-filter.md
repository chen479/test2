# 第9章 FIR滤波器设计

## 9.1 引言

有限冲激响应（FIR）数字滤波器的单位脉冲响应是有限长的，其系统函数只有零点（除原点外无极点），因此FIR滤波器总是稳定的，且可以设计成严格线性相位。这些优点使FIR滤波器在通信、图像处理、音频处理等领域得到广泛应用。本章介绍FIR滤波器的线性相位条件、常用设计方法（窗函数法、频率采样法、等波纹最优设计）以及MATLAB实现。

---

## 9.2 FIR滤波器的特点与线性相位条件

### 9.2.1 FIR滤波器的特点
- 单位脉冲响应 $h[n]$ 长度为 $N$，即 $h[n] = 0$ 对 $n<0$ 和 $n \ge N$。
- 系统函数 $H(z) = \sum_{n=0}^{N-1} h[n] z^{-n}$，只有零点，无极点（除 $z=0$ 外）。
- 总是稳定的，且易于实现线性相位。
- 可用FFT快速实现卷积。
- 设计方法成熟，但达到相同过渡带宽度所需阶数通常比IIR高。

### 9.2.2 线性相位条件
FIR滤波器具有线性相位的条件是：单位脉冲响应满足对称或反对称。

- **对称情况**：$h[n] = h[N-1-n]$（偶对称）
- **反对称情况**：$h[n] = -h[N-1-n]$（奇对称）

根据 $N$ 为奇或偶，可分为四种类型：

| 类型 | 对称性 | N | 适用频率响应 |
|------|--------|---|--------------|
| I型 | 偶对称 | 奇数 | 所有滤波器（低通、高通、带通、带阻） |
| II型 | 偶对称 | 偶数 | 低通、带通（$H(\pi)=0$，不能用做高通、带阻） |
| III型 | 奇对称 | 奇数 | 带通、微分器、希尔伯特变换器（$H(0)=0$，$H(\pi)=0$） |
| IV型 | 奇对称 | 偶数 | 高通、带通、微分器、希尔伯特变换器（$H(0)=0$） |

线性相位意味着群延迟为常数 $\tau = (N-1)/2$，信号通过滤波器后波形形状不变，仅延迟 $\tau$。

## 9.3 窗函数设计法

窗函数法是设计FIR滤波器最简单直观的方法，其基本思想是从理想频率响应出发，通过截断和平滑得到实际脉冲响应。

### 9.3.1 设计步骤
1. 根据技术指标确定理想滤波器的频率响应 $H_d(e^{j\omega})$（如理想低通）。
2. 求理想单位脉冲响应 $h_d[n]$（IDTFT）：
   $$
   h_d[n] = \frac{1}{2\pi} \int_{-\pi}^{\pi} H_d(e^{j\omega}) e^{j\omega n} d\omega
   $$
   对于理想低通（截止频率 $\omega_c$）：
   $$
   h_d[n] = \frac{\sin(\omega_c n)}{\pi n}, \quad -\infty < n < \infty
   $$
3. 选择窗函数 $w[n]$，长度为 $N$，得到实际脉冲响应：
   $$
   h[n] = h_d[n] w[n], \quad n = 0,1,\dots,N-1
   $$
   注意 $h_d[n]$ 需移位以保持因果性：取 $h_d\left[n - \frac{N-1}{2}\right]$ 再乘以窗。

4. 验证设计结果，若不满足指标，则增加 $N$ 或换用其他窗函数。

### 9.3.2 常用窗函数
| 窗函数名称 | 主瓣宽度（π/N倍数） | 旁瓣峰值衰减（dB） | 最小阻带衰减（dB） |
|------------|---------------------|---------------------|--------------------|
| 矩形窗 | 2 | -13 | -21 |
| 汉宁窗 | 4 | -31 | -44 |
| 汉明窗 | 4 | -41 | -53 |
| 布莱克曼窗 | 6 | -57 | -74 |
| 凯泽窗（β可调） | 可调 | 可调 | 可调 |

凯泽窗由零阶修正贝塞尔函数定义，可通过调整参数 $\beta$ 在主瓣宽度和旁瓣衰减之间折衷。

### 9.3.3 设计实例
设计一个低通滤波器，指标：$\omega_p = 0.2\pi$，$\omega_s = 0.3\pi$，$R_p \le 0.25$ dB，$R_s \ge 50$ dB。

由过渡带宽 $\Delta\omega = 0.1\pi$，根据凯泽窗经验公式：
$$
\Delta\omega = \frac{8\pi}{N} \Rightarrow N \approx \frac{8\pi}{0.1\pi} = 80
$$
或使用 `kaiserord` 函数直接估算阶数。用汉明窗也可近似达到50dB阻带衰减，但需适当选择 $N$。

## 9.4 频率采样设计法

频率采样法直接在频域对理想频率响应进行采样，然后通过IDFT得到脉冲响应。

### 9.4.1 基本思想
在 $\omega \in [0, 2\pi)$ 上等间隔取 $N$ 个点 $\omega_k = 2\pi k/N$，设期望的频率响应样本为 $H_d[k] = H_d(e^{j\omega_k})$。则脉冲响应为：
$$
h[n] = \frac{1}{N} \sum_{k=0}^{N-1} H_d[k] e^{j\frac{2\pi}{N}kn}, \quad n = 0,1,\dots,N-1
$$
这样得到的 $h[n]$ 的DFT正好等于 $H_d[k]$，但在采样点之间的实际频率响应可能偏离理想值，尤其在过渡带附近出现振荡（吉布斯现象）。

### 9.4.2 优化措施
为减小振荡，可在过渡带设置一个或多个**过渡点**（值为0~1之间的优化值），通过优化这些点的值使阻带衰减最大。常用方法有线性规划、Remez交换算法等。

### 9.4.3 设计步骤
1. 选择 $N$，确定采样频率点 $\omega_k = 2\pi k/N$。
2. 根据理想滤波器类型设定 $H_d[k]$（注意线性相位约束）。
3. 若需要优化，在过渡带引入一个或多个过渡点变量，用优化算法确定其值。
4. 计算IDFT得到 $h[n]$。
5. 验证性能，若不满足则调整 $N$ 或过渡点数量。

## 9.5 等波纹最优设计（Parks-McClellan算法）

窗函数法和频率采样法无法精确控制通带和阻带的波纹幅度，而**等波纹最优设计**（又称Parks-McClellan算法，基于Chebyshev逼近理论）可以实现在通带和阻带内波纹幅度相等的优化设计，使最大误差最小化。这是目前最常用的FIR设计方法。

### 9.5.1 设计思想
将设计问题转化为加权Chebyshev逼近问题：寻找一组系数 $h[n]$ 使加权逼近误差
$$
E(\omega) = W(\omega) \left[ H_d(e^{j\omega}) - H(e^{j\omega}) \right]
$$
的最大绝对值最小化，其中 $W(\omega)$ 为加权函数，用于平衡通带和阻带的误差。

### 9.5.2 Remez交换算法
该算法通过迭代在频率点集上交换极值点，最终得到满足等波纹准则的最优解。MATLAB中函数 `firpm` 实现此算法。

**调用格式**：
```matlab
h = firpm(N, f, a, w)
```
- `N`：滤波器阶数（脉冲响应长度 = N+1）
- `f`：归一化频率边界向量（0~1，1对应 $\pi$）
- `a`：对应频率点的期望幅值
- `w`：各频段的加权向量（可选）

**示例**：设计一个低通滤波器，通带0~0.2，阻带0.25~0.5，通带波纹1，阻带波纹10（即阻带权重为10）。
```matlab
h = firpm(50, [0 0.2 0.25 0.5], [1 1 0 0], [1 10]);
freqz(h,1);
```

### 9.5.3 阶数估计
可用 `firpmord` 函数根据指标估算所需阶数：
```matlab
[n, fo, ao, w] = firpmord([0.2 0.25]*(2), [1 0], [0.01 0.001], 2);
h = firpm(n, fo, ao, w);
```
其中 `[0.2 0.25]` 是通带和阻带截止频率（归一化到1），`[1 0]` 是期望幅值，`[0.01 0.001]` 是通带和阻带波纹（绝对值），`2` 是采样频率（归一化后通常写2表示 $\pi$）。

## 9.6 IIR与FIR滤波器的比较

| 特性 | IIR | FIR |
|------|-----|-----|
| 单位脉冲响应 | 无限长 | 有限长 |
| 稳定性 | 需设计保证 | 总是稳定 |
| 线性相位 | 不易实现 | 容易实现（对称结构） |
| 设计方法 | 基于模拟原型 | 窗函数、频率采样、等波纹 |
| 阶数 | 较低 | 较高 |
| 运算量 | 较小 | 较大 |
| 有限字长效应 | 敏感（可能振荡） | 不敏感 |
| 适用场合 | 对相位不敏感、要求计算量小 | 要求线性相位、允许较高阶数 |

1. 分别用矩形窗、汉明窗和布莱克曼窗设计一个长度为 $N=31$ 的低通滤波器，截止频率 $\omega_c = 0.4\pi$。计算并比较它们的幅频响应（用MATLAB绘制），观察主瓣宽度和旁瓣衰减。

2. 用窗函数法设计一个带通滤波器，指标：通带 $0.3\pi \le \omega \le 0.5\pi$，过渡带宽 $\Delta\omega = 0.1\pi$，阻带衰减 $\ge 40$ dB。选择合适的窗函数和阶数，写出设计步骤。

3. 用频率采样法设计一个长度为 $N=15$ 的线性相位低通滤波器，截止频率 $\omega_c = 0.4\pi$（对应采样点索引 $k=0,1,...,7$ 置1，其余置0）。计算其脉冲响应，并分析实际频率响应与理想响应的偏差。

4. 用Parks-McClellan算法设计一个高通滤波器，指标：阻带 $0 \le \omega \le 0.3\pi$，阻带衰减 $\ge 40$ dB；通带 $0.5\pi \le \omega \le \pi$，通带波纹 $\le 0.1$ dB。用 `firpmord` 估算阶数，并绘制滤波器的幅频响应和脉冲响应。

5. 设计一个数字微分器，要求在 $0 \le \omega \le 0.8\pi$ 范围内幅度响应为 $|H(e^{j\omega})| = \omega$，用等波纹设计实现（注意 III/IV型对称性）。

6. 简述窗函数法中吉布斯现象的成因，以及如何通过窗函数抑制它。

## 9.9 本章小结

FIR滤波器因具有严格线性相位、稳定等优点而广泛应用。窗函数法简单直观，适合快速设计；频率采样法在频域直接处理，可优化过渡点；等波纹最优设计（Parks-McClellan）能精确控制波纹，是最优设计。掌握这些方法对实际工程应用至关重要。下一章将讨论数字信号处理中的有限字长效应。
