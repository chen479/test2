# 第8章 IIR滤波器设计

## 8.1 引言

无限冲激响应（IIR）数字滤波器的特点是系统函数既有极点又有零点，单位脉冲响应无限长。与FIR滤波器相比，IIR滤波器可以用较低的阶数实现较陡的过渡带，但相位特性一般为非线性。本章介绍IIR滤波器的经典设计方法，这些方法通常基于模拟滤波器原型，通过变换得到数字滤波器。

---

## 8.2 数字滤波器设计的技术指标

设计数字滤波器前，需要明确技术指标，通常以幅频响应的容限图给出：

- **通带截止频率** $\omega_p$：通带边缘频率。
- **阻带截止频率** $\omega_s$：阻带起始频率。
- **通带最大衰减** $R_p$（dB）：通带内允许的最大幅值变化，通常定义为 $R_p = 20\log_{10} \left| \frac{H(e^{j0})}{H(e^{j\omega_p})} \right|$。
- **阻带最小衰减** $R_s$（dB）：阻带内允许的最小衰减，通常定义为 $R_s = -20\log_{10} \left| \frac{H(e^{j\omega_s})}{H(e^{j0})} \right|$。

对于低通滤波器，$\omega_p < \omega_s$；高通、带通、带阻的指标类似，但需定义多个边界频率。

---

## 8.3 模拟原型滤波器

IIR数字滤波器设计通常从模拟滤波器原型出发，将其变换为数字滤波器。常用的模拟原型滤波器有：

### 8.3.1 巴特沃斯滤波器
特点：通带内最平坦，单调下降，无波纹。其幅度平方函数为：
$$
|H_a(j\Omega)|^2 = \frac{1}{1 + (\Omega/\Omega_c)^{2N}}
$$
其中 $N$ 为阶数，$\Omega_c$ 为3dB截止频率。巴特沃斯滤波器的设计由给定的 $\Omega_p$、$\Omega_s$、$R_p$、$R_s$ 确定阶数 $N$ 和 $\Omega_c$。

### 8.3.2 切比雪夫I型滤波器
特点：通带内等波纹，阻带单调。幅度平方函数为：
$$
|H_a(j\Omega)|^2 = \frac{1}{1 + \varepsilon^2 T_N^2(\Omega/\Omega_p)}
$$
其中 $T_N(x)$ 为N阶切比雪夫多项式，$\varepsilon$ 控制通带波纹。

### 8.3.3 切比雪夫II型滤波器
特点：通带单调，阻带等波纹。又称为逆切比雪夫滤波器。

### 8.3.4 椭圆滤波器
特点：通带和阻带均为等波纹，过渡带最陡，但相位非线性严重。其幅度平方函数涉及雅可比椭圆函数。

设计时根据指标选择合适类型，通常巴特沃斯相位特性最好，椭圆滤波器阶数最低。

---

## 8.4 冲激响应不变法

### 8.4.1 基本原理
冲激响应不变法的核心是使数字滤波器的单位脉冲响应 $h[n]$ 等于模拟滤波器 $h_a(t)$ 的采样：
$$
h[n] = T_d h_a(nT_d)
$$
其中 $T_d$ 为采样间隔。对应的变换关系为：
$$
H(z) = \sum_{n=-\infty}^{\infty} h[n] z^{-n} = \sum_{n=-\infty}^{\infty} T_d h_a(nT_d) z^{-n}
$$
从s平面到z平面的映射为 $z = e^{sT_d}$，即将s平面的每一条宽度为 $2\pi/T_d$ 的水平带映射到整个z平面。

### 8.4.2 设计步骤
1. 根据数字指标 $\omega_p$、$\omega_s$ 转换为模拟指标：$\Omega_p = \omega_p / T_d$，$\Omega_s = \omega_s / T_d$。
2. 设计模拟滤波器 $H_a(s)$（如巴特沃斯、切比雪夫等）。
3. 将 $H_a(s)$ 展开为部分分式：$H_a(s) = \sum_{i=1}^{N} \frac{A_i}{s - s_i}$。
4. 得到数字滤波器：
   $$
   H(z) = \sum_{i=1}^{N} \frac{T_d A_i}{1 - e^{s_i T_d} z^{-1}}
   $$

### 8.4.3 优缺点
- **优点**：时域特性逼近好，线性相位特性保持（若模拟滤波器有线性相位）。
- **缺点**：会产生频谱混叠，因此只适用于限带模拟滤波器（如低通、带通），不适用于高通和带阻。

---

## 8.5 双线性变换法

### 8.5.1 基本原理
双线性变换法通过代数变换将整个s平面映射到z平面，克服了混叠问题。变换公式为：
$$
s = \frac{2}{T_d} \cdot \frac{1 - z^{-1}}{1 + z^{-1}}
$$
或
$$
z = \frac{1 + sT_d/2}{1 - sT_d/2}
$$
该变换将s平面的虚轴 $s=j\Omega$ 映射为z平面的单位圆，且是单值映射，无混叠。

### 8.5.2 频率预畸变
由于变换的非线性，模拟频率 $\Omega$ 与数字频率 $\omega$ 的关系为：
$$
\Omega = \frac{2}{T_d} \tan\left(\frac{\omega}{2}\right)
$$
设计时必须进行预畸变：将数字指标 $\omega$ 按上式转换为模拟指标 $\Omega$，设计模拟滤波器后再用双线性变换得到 $H(z)$。

### 8.5.3 设计步骤
1. 给定数字滤波器的技术指标 $\omega_p$、$\omega_s$、$R_p$、$R_s$。
2. 选择 $T_d$（通常取1或2），进行频率预畸变：
   $$
   \Omega_p = \frac{2}{T_d} \tan\left(\frac{\omega_p}{2}\right), \quad
   \Omega_s = \frac{2}{T_d} \tan\left(\frac{\omega_s}{2}\right)
   $$
3. 设计模拟滤波器 $H_a(s)$ 满足预畸变后的指标。
4. 用双线性变换得到数字滤波器 $H(z) = H_a\left( \frac{2}{T_d} \cdot \frac{1 - z^{-1}}{1 + z^{-1}} \right)$。

### 8.5.4 优缺点
- **优点**：无混叠，可设计任何类型滤波器（低通、高通、带通、带阻）。
- **缺点**：频率变换是非线性的，使得数字频率与模拟频率呈非线性关系，可能使设计复杂（但对幅频响应要求严格的场合，可通过预畸变精确控制边界频率）。

---

## 8.6 频率变换设计高通、带通、带阻滤波器

通过双线性变换，可以从模拟低通原型直接设计高通、带通、带阻滤波器。常用方法有两种：

### 8.6.1 先模拟频率变换，再双线性变换
1. 将数字滤波器的技术指标通过预畸变转换为模拟低通原型指标。
2. 设计模拟低通原型 $H_{LP}(s)$。
3. 对 $H_{LP}(s)$ 进行模拟频率变换（如低通→高通、低通→带通），得到所需类型的模拟滤波器 $H_a(s)$。
4. 对 $H_a(s)$ 应用双线性变换得到数字滤波器。

### 8.6.2 直接数字域频率变换
对于已经设计好的数字低通滤波器，可通过数字域频率变换（如 $z^{-1} \to -z^{-1}$ 实现低通→高通）得到其他类型。但这种方法需谨慎处理稳定性。

MATLAB中，函数 `lp2lp`、`lp2hp`、`lp2bp`、`lp2bs` 可实现模拟频率变换。

## 8.7 IIR滤波器设计的MATLAB实现

MATLAB信号处理工具箱提供了丰富的IIR滤波器设计函数：

| 函数名 | 说明 |
|--------|------|
| `butter` | 巴特沃斯滤波器设计（可低通、高通、带通、带阻） |
| `cheby1` | 切比雪夫I型滤波器设计 |
| `cheby2` | 切比雪夫II型滤波器设计 |
| `ellip` | 椭圆滤波器设计 |
| `bilinear` | 双线性变换 |
| `impinvar` | 冲激响应不变法（需先将模拟滤波器转换为零极点形式） |

**示例**：用双线性变换法设计一个巴特沃斯低通数字滤波器，指标为：$\omega_p=0.2\pi$，$\omega_s=0.3\pi$，$R_p=0.5$ dB，$R_s=40$ dB。

```matlab
% 数字指标
wp = 0.2*pi; ws = 0.3*pi;
Rp = 0.5; Rs = 40;

% 频率预畸变（取 T=1）
Wp = 2 * tan(wp/2);
Ws = 2 * tan(ws/2);

% 设计模拟巴特沃斯滤波器
[n, Wn] = buttord(Wp, Ws, Rp, Rs, 's');
[bs, as] = butter(n, Wn, 's');

% 双线性变换得到数字滤波器
[b, a] = bilinear(bs, as, 1); % 采样频率为 1/T = 1
freqz(b, a); % 绘制频率响应
```

## 8.10 本章小结

IIR滤波器设计主要基于模拟滤波器原型，通过冲激响应不变法或双线性变换法转换为数字滤波器。冲激响应不变法时域特性好，但有混叠限制；双线性变换法无混叠，但频率非线性。实际中常采用双线性变换结合频率预畸变。MATLAB提供了便捷的设计工具。掌握IIR设计对构建信号处理系统至关重要。下一章将介绍FIR滤波器设计方法。
