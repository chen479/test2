# 第2章 离散时间信号与系统

## 2.1 离散时间信号（序列）

### 2.1.1 序列的定义
离散时间信号（简称序列）是指在离散时间点上有定义的信号，通常表示为 x[n]，其中 n  为整数（时间序号）。序列可以是自然产生的（如经济数据、人口统计），也可以通过对连续时间信号采样得到。

### 2.1.2 常用基本序列

1. **单位脉冲序列（单位采样序列）**  
   $$
   \delta[n] = \begin{cases}
   1, & n = 0 \\
   0, & n \neq 0
   \end{cases}
   $$

2. **单位阶跃序列**  
   $$
   u[n] = \begin{cases}
   1, & n \geq 0 \\
   0, & n < 0
   \end{cases}
   $$

3. **矩形序列**  
   $$
   R_N[n] = \begin{cases}
   1, & 0 \leq n \leq N-1 \\
   0, & \text{其他}
   \end{cases}
   $$

4. **实指数序列**  
   $$
   x[n] = a^n u[n] \quad (a \text{为实数})
   $$

5. **正弦序列**  
   $$
   x[n] = A \sin(\omega n + \phi)
   $$
   其中 $\omega$ 为数字角频率（弧度/样本），$\phi$ 为初相。

6. **复指数序列**  
   $$
   x[n] = e^{(\sigma + j\omega)n} = e^{\sigma n} e^{j\omega n}
   $$

### 2.1.3 序列的运算
- **移位**：$x[n - k]$（右移 $k$ 位，$k>0$）
- **反转**：$x[-n]$
- **和**：$y[n] = x_1[n] + x_2[n]$
- **积**：$y[n] = x_1[n] \cdot x_2[n]$
- **累加**：$y[n] = \sum_{k=-\infty}^{n} x[k]$

---

## 2.2 离散时间系统的分类

离散时间系统将输入序列 $x[n]$ 映射为输出序列 $y[n]$，记为 $y[n] = T\{x[n]\}$。

### 2.2.1 线性系统
系统满足叠加原理：
- 可加性：$T\{x_1[n] + x_2[n]\} = T\{x_1[n]\} + T\{x_2[n]\}$
- 齐次性：$T\{a x[n]\} = a T\{x[n]\}$（$a$ 为常数）

### 2.2.2 时不变系统
若输入延迟 $k$ 个单位，输出也延迟相同单位：
$$
y[n] = T\{x[n]\} \Rightarrow T\{x[n-k]\} = y[n-k]
$$

### 2.2.3 因果系统
系统的输出只取决于当前及过去的输入，与未来输入无关。即：若 $x_1[n] = x_2[n]$ 对于所有 $n < n_0$，则对于 $n < n_0$ 有 $y_1[n] = y_2[n]$。

### 2.2.4 稳定系统
有界输入产生有界输出（BIBO稳定）。即若 $|x[n]| \leq M_x < \infty$，则 $|y[n]| \leq M_y < \infty$。

---

## 2.3 线性时不变系统与卷积和

### 2.3.1 单位脉冲响应
线性时不变（LTI）系统可由单位脉冲响应 $h[n]$ 完全描述：
$$
h[n] = T\{\delta[n]\}
$$

### 2.3.2 卷积和
对于任意输入 $x[n]$，LTI 系统的输出为：
$$
y[n] = x[n] * h[n] = \sum_{k=-\infty}^{\infty} x[k] h[n-k]
$$
卷积运算满足交换律、结合律和分配律。

### 2.3.3 卷积的计算方法
- 解析法（利用求和公式）
- 图形法（翻转、移位、相乘、求和）
- MATLAB 数值计算（`conv` 函数）

---

## 2.4 线性常系数差分方程

许多离散时间系统可用线性常系数差分方程描述：
$$
\sum_{k=0}^{N} a_k y[n-k] = \sum_{k=0}^{M} b_k x[n-k]
$$
其中 $a_k, b_k$ 为常数，$a_0 \neq 0$。

### 2.4.1 递推求解
若系统是因果的，且已知初始条件，可由差分方程递推求出 $y[n]$。

### 2.4.2 与卷积的关系
对于 LTI 系统，差分方程的解就是单位脉冲响应与输入的卷积。通过 z 变换可方便地求解。

## 2.5 习题

1. 判断下列序列是否为周期信号，若是则求周期：
   (a) $x[n] = \cos\left(\frac{2\pi}{5}n\right)$
   (b) $x[n] = e^{j\left(\frac{3\pi}{4}n\right)}$
   (c) $x[n] = \sin(3n)$

2. 已知序列 $x[n] = \{1,2,3,4\}$（$n=0,1,2,3$），$h[n] = \{1,1,1\}$（$n=0,1,2$），求卷积 $y[n] = x[n] * h[n]$，并画出结果。

3. 判断下列系统的线性、时不变性、因果性、稳定性：
   (a) $y[n] = n x[n]$
   (b) $y[n] = x[n] + x[n-1]$
   (c) $y[n] = x[-n]$

4. 某 LTI 系统的单位脉冲响应为 $h[n] = (0.5)^n u[n]$，求输入 $x[n] = u[n]$ 时的输出 $y[n]$。
