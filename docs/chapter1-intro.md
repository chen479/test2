# 第1章 绪论

## 1.1 数字信号处理概述

数字信号处理（Digital Signal Processing，DSP）是利用计算机或专用数字硬件，对离散时间信号进行分析、变换、滤波、检测、估计、识别等处理的学科。它与模拟信号处理相比，具有精度高、灵活性大、可靠性强、易于大规模集成等优点。

### 1.1.1 信号与系统的基本概念
- **信号**：携带信息的物理量随时间或空间的变化。按自变量和函数值的取值特点，可分为：
    - 连续信号（时间和幅度均连续）
    - 时域离散信号（在时间上离散取值，幅度连续）
    - 幅度离散信号（时间连续，幅度离散取值）
    - 数字信号（时间和幅度均离散）
- **系统**：对输入信号进行特定变换或运算的物理装置或算法。

### 1.1.2 数字信号处理的发展历程
- 20世纪60年代：快速傅里叶变换（FFT）的提出，使实时处理成为可能。
- 20世纪70-80年代：数字信号处理器（DSP芯片）的出现，推动了通信、音频等领域的发展。
- 21世纪至今：与人工智能、物联网、5G等新兴技术深度融合。

---

## 1.2 数字信号处理系统的组成

一个典型的数字信号处理系统由以下几个部分组成：

   模拟信号 → 抗混叠滤波器 → A/D转换器 → 数字信号处理器 → D/A转换器 → 平滑滤波器 → 模拟信号

- **抗混叠滤波器**：限制输入信号的带宽，避免采样时发生频谱混叠。
- **A/D转换器**：将模拟信号转换为数字信号，包括采样、量化和编码。
- **数字信号处理器**：执行核心算法（如FIR/IIR滤波、FFT、相关运算等）。
- **D/A转换器**：将处理后的数字信号恢复为模拟信号。
- **平滑滤波器**：滤除D/A转换产生的高频镜像分量。

## 1.3 数字信号处理的应用领域

数字信号处理已渗透到几乎所有工程和科学领域，典型应用包括：

| 领域 | 典型应用 |
|:-----|:---------|
| **通信** | 调制解调、信道均衡、语音编码、软件无线电 |
| **音频处理** | 音频压缩（MP3）、均衡器、回声消除、语音识别 |
| **图像处理** | 图像压缩（JPEG）、增强、复原、特征提取 |
| **生物医学** | 心电图（ECG）去噪、脑电图（EEG）分析、医学成像 |
| **雷达/声呐** | 脉冲压缩、波束形成、目标检测与跟踪 |
| **工业控制** | 振动分析、故障诊断、过程控制 |
| **消费电子** | 手机、数码相机、智能音箱中的信号处理算法 |
