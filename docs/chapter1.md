# 第1章 离散时间信号与系统

## 1.1 离散时间信号

离散时间信号是指在时间上取离散值的信号，通常用序列 \(x[n]\) 表示，其中 \(n\) 为整数。

### 常见的离散时间信号
- **单位脉冲序列**：\(\delta[n] = \begin{cases} 1, & n=0 \\ 0, & n \neq 0 \end{cases}\)
- **单位阶跃序列**：\(u[n] = \begin{cases} 1, & n \geq 0 \\ 0, & n < 0 \end{cases}\)
- **正弦序列**：\(x[n] = \sin(\omega n + \phi)\)

## 1.2 离散时间系统

离散时间系统将输入序列 \(x[n]\) 映射为输出序列 \(y[n]\)，记为 \(y[n] = T\{x[n]\}\)。

### 线性时不变系统
如果系统满足**线性**和**时不变性**，则称为线性时不变（LTI）系统。LTI 系统可由单位脉冲响应 \(h[n]\) 完全表征，输出为输入与 \(h[n]\) 的卷积：
\[
y[n] = x[n] * h[n] = \sum_{k=-\infty}^{\infty} x[k] h[n-k]
\]
